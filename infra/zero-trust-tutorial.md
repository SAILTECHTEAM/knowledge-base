# Cloudflare Zero Trust Connection Guide

This guide will help you connect your device desktop (Windows, Mac, Linux) or mobile (iOS, Android) to Cloudflare Zero Trust.

## 1. Prerequisites

- **Access Policy Set Up:** Your admin should have set up access policies for your device or user group.

## 2. Desktop Devices

### a. Download and install WARP Client

- **Windows/Mac:** [Cloudflare WARP Download](https://cloudflarewarp.com/)
- **Linux:** [Cloudflare WARP Linux Docs](https://developers.cloudflare.com/cloudflare-one/connections/connect-devices/warp/)

### b. Configure Zero Trust

1. Open WARP client.
2. Go to **Preference > Account**.
3. Select **Login with Cloudflare Zero Trust**.
4. Enter team name: `your-team` or `your-team.cloudflareaccess.com`
5. Authenticate as prompted.
6. Click the **Connect** toggle

## 3. Mobile Devices (iOS / Android)

### a. Download WARP App

- **iOS:** [App Store - Cloudflare One Agent](https://apps.apple.com/us/app/cloudflare-one-agent/id6443476492)
- **Android:** [Google Play - Cloudflare One Agent](https://play.google.com/store/apps/details?id=com.cloudflare.cloudflareoneagent)

### b. Install and Cloudflare One Agent

1. Open the app.
2. Accept terms and permissions.

### c. Connect to Zero Trust

1. Tap the burger menu on the top right corner.
2. Select **Account > Login to Cloudflare Zero Trust**.
3. Enter team name: `your-team` or `your-team.cloudflareaccess.com`
4. Authenticate as prompted.
5. Tap **"Connect"** toggle.
